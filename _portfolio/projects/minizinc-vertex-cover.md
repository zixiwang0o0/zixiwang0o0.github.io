## MiniZinc Vertex Cover

This project models the vertex cover problem using MiniZinc. The goal is to
choose a small set of vertices so that every edge in a graph is covered by at
least one selected vertex.

### Focus

- Representing graph constraints clearly
- Encoding optimization objectives
- Testing small examples before scaling up
